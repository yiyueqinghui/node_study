node的优势

  • 1、性能高

npm包管理

  • 1、运行npm init，生成package.json文件，即包管理文件
  • 2、通过npm install 包名 (简写 npm i 包名) 安装依赖
  • 3、通过npm uninstall 包名 (简写 npm un 包名) 移除依赖
  • 4、通过npm update 包名 更新依赖
  • 5、通过npm install (简写 npm i) 安装package.json中的所有依赖

node的模块

1、全局模块

  • 定义：何时何地才能访问，不需要引用
  • process.env 环境变量
  • process.arg 传入的参数
  • __dirname

2、系统模块

  • 定义: 需要require(),但不需要单独下载
  • path: 用于处理文件路径和目录路径的实用工具
  • fs: 文件读写操作

3、自定义模块

  • 定义：require自己封装的模块
  • exports
  • module
  • require 如果有路径就去路径里找，没有路径的话就去node_modules里面找

HTTP模块

  • 服务器对象：http.createServer() 快速搭建服务器

GET请求

浏览器与服务器之间的传递的数据，称之为报文。
报文分为
头：各种信息，大小<=32k(这是http规定的)
体：数据，大小<2G

  • 数据是放在url里面进行传输的,传输的容量比较小，<32k

POST请求

  • 数据是放在报文体里面进行传输的,传输的容量比较大，<2G
